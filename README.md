# Groww FastAPI Trading Backend

A FastAPI-based backend for trading using Groww APIs.

## 🚀 Features
- Buy / Sell Orders
- Limit / Market Orders
- WebSocket Live Data
- Portfolio & Margin APIs
- Instrument-based segment detection
